# CLIP-SBR

## Environments

- **Python**: 3.10
- **PyTorch**: 1.12

### RecBole
To install RecBole, use the following command:
```bash
pip install recbole
```
### RecBole-GNN
To install RecBole-GNN, use the following command:
```bash
git clone https://github.com/RUCAIBox/RecBole-GNN
```
Copy the recbole_gnn folder to the root directory.
